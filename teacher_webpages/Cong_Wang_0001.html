Overview

I am interested in developing secure software systems with theoretical security / privacy guarantee and practical performance. My recent work has focused on data security (e.g., encrypted databases, multi-party encrypted analytics, proofs of remote storage, etc.), AI systems and security (e.g., red teaming for LLM, federated (un)learning, robust reinforcement learning, etc.), privacy-enhancing technologies (e.g., encrypted messaging and metadata privacy, anonymous communications, encrypted network middleboxes, etc.), blockchain and decentralised applications (e.g., decentralized storage auditing, on-chain/off-chain interoperability, consensus designs, etc.), and software security and systems (e.g., hardware-assisted security, anti-fuzzing techniques, boosting performance of deep learning clusters, etc.). Please refer to the following links and the group website for more details.
